How to Release Fit

  1. Prep
    1. Update changes.txt
    2. Update readme.html
    3. Update .NET AssemblyInfo.cs version number

  2. Build Java and .NET versions
    1. "build release"
    2. Double-check release directories
    3. Rename file to include version number (e.g., fit-java-1.1.zip)
    4. Commit changes to CVS
    5. Tag CVS with name of release (e.g., Release1-1)

  3. Upload
    1. Upload files to ftp://upload.sourceforge.net/incoming (anonymous login)
    2. Register release with SourceForge at https://sourceforge.net/project/admin/editpackages.php?group_id=85646

  4. Update Website
    1. Change current release on WelcomeVisitors
    2. Update DownloadNow
    3. Update LatestChanges
